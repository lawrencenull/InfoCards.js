Infocards.js : The nice way to get information

Description

infocards.js is a customizable JS plugin which allows you to get related information about a specified query. It is powered by the awesome DuckDuckGo api, which respects your privacy.

Usage

new InfoCard(options)

options

query

The query you wish to get information about. Required

container

The DOM element where you wish to insert the info card. Required

classNames

Class names you wish to apply to differents DOM elements.

Supported values :

  • category-item
  • card
  • tabs

Usage :

classNames: {
    category-item: "card",
    tabs: "tabs"
}

horizontalScrolling

Enables horizontal scrolling support using the mousewheel for some elements.

Supported values :

  • false (default) : Disables horizontal scrolling
  • true : Enables horizontal scrolling
  • "string" : Enables horizontal scrolling for selectors specified inside the string

onEmpty(container)

Function called when no results are found.

onError(errorCode)

Function called when an error occurs.

errorCode
  • 0: 404 error
  • 1: Unimplemented
