Infocards.js : The nice way to get information

Description

infocards.js is a customizable JS plugin which allows you to get related information about a specified query. It is powered by the awesome DuckDuckGo api, which respects your privacy.

Usage

new InfoCard(options)

options

query

The query you wish to get information about. Required

container

The DOM element where you wish to insert the info card. Required
